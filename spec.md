# Specification

## Summary
**Goal:** Restore the application to the exact state of Version 12, reverting all changes made in versions 13 through 16.

**Planned changes:**
- Revert all frontend files to their Version 12 state
- Revert all backend files to their Version 12 state
- Remove any code or logic introduced in versions 13, 14, 15, or 16

**User-visible outcome:** The application runs exactly as it did at Version 12, with no changes from subsequent versions present.
